Система аутентификации


Техническое задание

  1. Для входа в систему Пользователю необходимо:
    1. указать E-mail/Логин и пароль
    2. нажать кнопку "Войти"
  2. После успешной аутентификации пользователь получает куки с сессией
  3. При неуспешной аутентификации отображаются допущенные ошибки
  4. E-mail можно указать в любом регистре, но допускается совпадение E-mail для разных пользователей с символами в разных регистрах

Задачи

  1. Написать API и UI автотесты на страницу входа Grey Box подходом
  2. Не забыть про возможность настройки (например, под разные среды и стенды)
  3. Для Java:
    1. Использовать JUnit5/TestNG
    2. Использовать RestAssured
    3. Использовать Selenium/Selenide
    4. Использовать Allure
    5. Использовать Maven/Gradle
  4. Подключение той или иной библиотеки к проекту автотестов аргументировать в комментариях: зачем, почему именно она
